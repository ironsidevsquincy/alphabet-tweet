Alphabet Tweet
==============

Requirements
-------------

 * Twitter4r >= 0.6.0 (http://twitter4r.rubyforge.org/)
 * Dict (http://www.caliban.org/ruby/ruby-dict.shtml)
 
Usage
-----

    $ ./alphabet_tweet.rb
    
Enter consumer and access secret when prompted
